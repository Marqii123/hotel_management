# hotel_management 

A new Flutter project for Front End Assestment.

1. Framework used
    - Flutter
    - Dart (Language)
    - Material Design (UI)

2. Installation & Setup

Prerequisites (if needed)
Ensure that Visual Studio Build Tools is installed
    - Install **MSVC v142** or **v143** (if using Windows)
    - Install **Windows 10/11 SDK**
    - Install **C++ CMake tools**

Run .exe file founded in \hotel_management\release\hotel_management.exe
or inside
\hotel_management\hotel_management\build\windows\x64\runner\Release\hotel_management.exe
